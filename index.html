Quick Sort Algorithm

How it works

  1. Choose a pivot element (now the first element)
  2. Partition the array:
    • i pointer: scans from left, moves right for elements ≤ pivot
    • j pointer: scans from right, moves left for elements > pivot
  3. Swap elements at i and j as needed
  4. Place pivot in its correct position
  5. Recursively sort left and right subarrays

Theoretical Analysis

  • Best Case: Pivot always splits array into two equal halves.
    Recurrence: T(n) = 2T(n/2) + O(n) ⇒ O(n log n)
  • Worst Case: Pivot is always the smallest/largest element (e.g., sorted array or all equal).
    Recurrence: T(n) = T(n-1) + O(n) ⇒ O(n²)
  • Average Case: O(n log n)
